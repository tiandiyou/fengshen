## 技术架构

### 后端架构
```
Spring Boot 3.2.2
├── Controller层 (REST API)
├── Service层 (业务逻辑)
├── Mapper层 (MyBatis-Plus)
└── Entity层 (数据模型)
```

### 前端架构
```
Cocos Creator 3.x + TypeScript
├── network/ (网络通信)
│   ├── HttpClient
│   ├── WebSocketClient
│   └── AuthService
├── data/ (数据管理)
│   ├── StorageManager
│   ├── CharacterConfig
│   ├── SkillConfig
│   └── ItemConfig
└── game/ (游戏逻辑)
    ├── battle/ (战斗系统)
    ├── character/ (角色系统)
    └── map/ (地图系统)
```
