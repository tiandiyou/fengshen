## 一、NES模拟器技术方案

### 1.1 推荐方案对比

| 方案 | 类型 | GitHub Stars | 性能 | 维护状态 | 推荐度 |
|------|------|-------------|------|---------|--------|
| **Nostalgist.js** | WASM封装 | 快速增长 | ⭐⭐⭐⭐⭐ | 活跃 | ⭐⭐⭐⭐⭐ |
| **JSNES** | 纯JavaScript | 3,500+ | ⭐⭐⭐⭐ | 活跃 | ⭐⭐⭐⭐ |
| **EmulatorJS** | Web前端 | 3,000+ | ⭐⭐⭐⭐⭐ | 活跃 | ⭐⭐⭐⭐ |
| **NES-Rust** | Rust→WASM | N/A | ⭐⭐⭐⭐⭐ | 活跃 | ⭐⭐⭐ |

### 1.2 最终推荐: **不使用模拟器,自行实现**

**理由:**
1. ✅ 完全掌控游戏逻辑
2. ✅ 避免版权问题(不依赖ROM)
3. ✅ 更好的跨平台适配(小程序)
4. ✅ 可自定义修改游戏机制
5. ✅ 学习价值更高

### 1.3 备选方案(仅参考)

如果确实需要模拟器:
- **Nostalgist.js**: 使用RetroArch fceumm核心,精度最高
- **JSNES**: 纯JavaScript实现,易于集成

## 四、游戏机制研究

### 4.1 角色系统

**四大主角:**
1. **哪吒** - 物理/力量型 (全程主角)
2. **小龙女** - 法术/治疗型 (第一章加入)
3. **杨戬** - 混合型战士 (第三章加入)
4. **姜子牙** - 辅助型军师 (第六章加入)

### 4.2 战斗系统

**回合制战斗:**
- 行动顺序基于敏捷值
- 指令: 攻击/法术/物品/逃跑
- 伤害公式: `(攻击 - 防御) × 随机系数(0.9-1.1)`
- 暴击率: 10% (1.5倍伤害)

### 4.3 法术系统

**小龙女(治疗系):**
- 提神术(12级): 单体治疗
- 摄魂术(43级): 全体伤害
- 复活术(57级): 复活队友

**杨戬(幻术系):**
- 幻啄鸡(24级): 单体伤害
- 幻雷公(35级): 全体伤害
- 幻石匠(41级): 石化控制

**姜子牙(辅助系):**
- 天罡术: 提升攻防
- 封魔术: 禁止敌人法术
- 凌鹰术: 快速传送

### 4.4 属性系统

```
体力(HP): 生命值
法力(MP): 法术消耗
攻击(ATK): 物理攻击力
防御(DEF): 物理防御力
敏捷(AGI): 行动顺序
体能(STA): 影响HP
```

## 六、NES ROM技术资料

### 6.1 ROM文件结构

```
iNES格式 (.nes):
├─ Header (16字节)
│  ├─ 字节4: PRG-ROM数量(16KB单位)
│  ├─ 字节5: CHR-ROM数量(8KB单位)
│  └─ 字节6-7: Mapper和镜像类型
├─ Trainer (可选, 512字节)
├─ PRG-ROM (程序代码)
└─ CHR-ROM (图形数据)
```

### 6.2 NES硬件规格

```
CPU: Ricoh 2A03 (基于6502)
  ├─ 频率: 1.79 MHz (NTSC)
  ├─ RAM: 2KB
  └─ 地址空间: 64KB

PPU (图形处理单元):
  ├─ 分辨率: 256×240
  ├─ VRAM: 2KB
  ├─ 精灵: 64个
  └─ 调色板: 4色×8组

APU (音频处理单元):
  ├─ 2个脉冲通道
  ├─ 1个三角波通道
  ├─ 1个噪声通道
  └─ 1个DMC通道
```

### 6.3 ROM分析工具

| 工具 | 用途 |
|------|------|
| **FCEUX** | 调试器、PPU查看器、Hex编辑器 |
| **Tile Layer Pro** | 图形编辑 |
| **YY-CHR** | CHR数据提取 |
| **Ghidra + NES插件** | 反汇编分析 |

## 七、微信小程序适配

### 7.1 游戏引擎选择

**Cocos Creator** 是最佳选择:
- ✅ 官方支持微信小游戏发布
- ✅ 一键构建小游戏包
- ✅ 自动适配API差异

### 7.2 关键适配点

```javascript
// 存储适配
Web: localStorage.setItem()
小程序: wx.setStorageSync()

// 音频适配
Web: Howler.js
小程序: wx.createInnerAudioContext()

// 网络适配
Web: WebSocket
小程序: wx.connectSocket()
```

### 7.3 小游戏限制

```
包体大小: 4MB (主包)
```
