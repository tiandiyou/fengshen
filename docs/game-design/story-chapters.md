# FC封神榜 - 章节流程文档

## 游戏章节总览

游戏共分为 **20个主要章节**,按照剧情发展依次解锁。

---

## 第一章: 讲理?复仇! (四海龙宫)

### 剧情概述
哪吒在陈塘关闹海,打伤龙王三太子,引发四海龙王水淹陈塘关。为平息事端,哪吒踏上征战四海龙宫之路。

### 主要流程
```
陈塘关(起点)
  ↓
陈塘村(搜刮装备)
  ↓
【南海龙宫】 → 击败南海龙王
  ↓
【西海龙宫】 → 击败西海龙王
  ↓
【北海龙宫】 → 击败北海龙王 → 获得【定海珠】
  ↓
捕鱼村(补给装备)
  ↓
使用【定海珠】消除海底旋涡
  ↓
【新月岛】 → 击败小龙女
  ↓
【东海龙宫】 → 击败东海龙王
  ↓
【剧情】哪吒自尽,小龙女加入队伍
```

### 关键地点
| 地点 | 类型 | 重要物品 | BOSS |
|------|------|---------|------|
| 陈塘关 | 村庄 | 初始装备 | - |
| 南海龙宫 | 地下迷宫 | 长剑 | 南海龙王 |
| 西海龙宫 | 地下迷宫 | - | 西海龙王 |
| 北海龙宫 | 地下迷宫 | 定海珠 | 北海龙王 |
| 捕鱼村 | 村庄 | 新装备 | - |
| 新月岛 | 地下迷宫 | - | 小龙女 |
| 东海龙宫 | 地下迷宫 | - | 东海龙王 |

### 推荐等级
- 南海龙宫: Lv3-5
- 西海龙宫: Lv5-7
- 北海龙宫: Lv7-9
- 东海龙宫: Lv9-11

## 第二章: 小龙女 (地狱之路)

### 剧情概述
哪吒为赎罪自尽后,灵魂下到地狱,必须在十殿阎罗中通过重重考验,方能重生。

### 主要流程
```
【奈河村】(地狱入口)
  ↓
【地狱第一殿】 → 击败秦广王
  ↓
【地狱第二殿】 → 击败楚江王
  ↓ (石泉谷隐藏地点: 无限药草)
【地狱第三殿】 → 击败宋帝王
  ↓
【地狱第四殿】 → 击败五宫王(会催眠)
  ↓
【地狱第五殿】 → 击败森罗王(会混乱)
  ↓
【地狱第六殿】 → 击败卞城王
  ↓
【地狱第七殿】 → 击败泰山王
  ↓
【地狱第八殿】 → 击败都市王
  ↓
【地狱第九殿】 → 击败平等王 → 获得【定神珠】
  ↓ (必须立即使用定神珠,否则走路掉血)
【地狱第十殿】 → 击败转轮王
  ↓
【剧情】哪吒重生
```

### 十殿阎罗属性
| 阎王 | 推荐等级 | 特殊技能 | 战术要点 |
|------|---------|---------|---------|
| 秦广王 | Lv13 | 无 | 硬拼即可 |
| 楚江王 | Lv14 | 高回避 | 需要小龙女法术支援 |
| 宋帝王 | Lv15 | 高攻击,低命中 | 易打出连击 |
| 五宫王 | Lv16 | 催眠法术 | 携带参须解毒 |
| 森罗王 | Lv17 | 混乱法术 | 保持HP充足 |
| 卞城王 | Lv18 | - | - |
| 泰山王 | Lv19 | - | - |
| 都市王 | Lv20 | - | - |
| 平等王 | Lv21 | - | 获得定神珠 |
| 转轮王 | Lv22 | - | 最终BOSS |

### 隐藏地点
- **石泉谷**: 在第二殿和第三殿之间,原地不动几秒即可进入,内有无限药草

## 第三章: 重生 (哪吒复活)

### 剧情概述
哪吒通过地狱考验,在莲花中重生,踏上新的征程。

### 主要流程
```
【平安村】(重生之地)
  ↓
【玉泉山】 → 获得杨戬情报
  ↓
【逸仙亭】 → 站在神龛前恢复全员状态
  ↓
【森林神木】 → 初遇杨戬
  ↓
回玉泉山找杨戬师父 → 获得玉佩
  ↓
回森林神木 → 杨戬加入队伍
  ↓
【九龙镇】 → 获得四大恶人情报
  ↓ (先去岛上洞窟和小童对话)
【五龙山】 → 文殊天尊 → 获得【遁龙桩】
  ↓
【九龙岛】 → 使用遁龙桩 → 击败四大恶人
  ↓
回九龙镇 → 断桥修复
```

### 新角色加入
**杨戬** - 三眼战神
- 加入等级: 通常Lv18-20
- 定位: 混合型战士,拥有幻术攻击
- 初始技能: 无
- 学习技能:
  - Lv24: 幻啄鸡, 映像术
  - Lv25: 幻毒蚣
  - Lv30: 跳跃术
  - Lv35: 幻雷公

### 四大恶人战术
- **必须先用遁龙桩**破除防护网,否则攻击无效
- 逐个击破,减少敌人数量
- 杨戬需要练到Lv25学会幻毒蚣

## 第四章: 花狐貂 (女儿村)

### 剧情概述
队伍前往佳东镇,遭遇花狐貂,小龙女失踪。

### 主要流程
```
【佳东镇】 → 获得魔家四将情报
  ↓
【后山窟】 → 击败花狐貂
  ↓ (小龙女失踪)
【女儿村】 → 获得捆仙绳情报
  ↓
【清峰山】 → 清虚真君
  ↓ 获得【夜明珠】和钻心钉情报
【暗黑洞窟】 → 使用夜明珠照明
  ↓ 最深处宝箱 → 获得【捆仙绳】
【女人国】 → 使用捆仙绳 → 击败女王
```

## 第五章: 魔家四将

### 剧情概述
队伍与魔家四将决战,杨戬染病,需要寻找仙药救治。

### 主要流程
```
【佳梦关】 → 使用钻心钉 → 击败魔家四将
  ↓ (战斗中小龙女归队,杨戬染病)
【西歧城】 → 与姬昌交谈
  ↓ 在水池获得【神木桨】
【芙冰村】 → 将神木桨给船夫 → 造船
  ↓
【海中香谢居】 → 百草仙子 → 获得【雪莲】
  ↓
使用雪莲救治杨戬
  ↓
【玉泉山】 → 获得火云洞情报
  ↓
【火云洞】 → 三圣大师 → 获得【丹药】
  ↓
回西歧城 → 将丹药投入井中 → 疫病清除
```

## 隐藏要素

### 秘籍系统(原版游戏)
- **无敌+穿墙+秒杀**: 上+选择+开始+B
- 建议复刻版**不保留**此秘籍,或作为Debug模式

### 隐藏地点
1. **石泉谷**: 地狱第二殿附近
2. **逸仙亭**: 玉泉山附近,可恢复全员状态
3. 各章节隐藏宝箱

### 隐藏装备
- 轩辕剑(需要特殊条件)
- 四不像(姜子牙专属坐骑)
- 哮天犬(杨戬专属)
