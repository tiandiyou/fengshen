## 三、战斗系统

### 3.1 战斗流程
```
1. 遭遇敌人(随机遇敌)
   ↓
2. 显示战斗界面(第一人称视角)
   ↓
3. 计算行动顺序(基于敏捷)
   ↓
4. 玩家选择指令
   ├─ 攻击: 普通物理攻击
   ├─ 法术: 使用已学技能
   ├─ 物品: 使用消耗品
   └─ 逃跑: 尝试逃离战斗
   ↓
5. 执行回合(按行动顺序)
   ↓
6. 敌人行动
   ↓
7. 判断战斗结束
   ├─ 敌人全灭 → 获得经验/金币/物品
   ├─ 我方全灭 → 游戏结束
   └─ 继续战斗 → 回到步骤3
```

### 3.2 伤害计算公式

#### 物理伤害
```java
// 基础伤害 = 攻击力 - 防御力
int baseDamage = attacker.getAttack() - defender.getDefense();

// 随机浮动(0.9 ~ 1.1倍)
float randomFactor = 0.9f + Math.random() * 0.2f;

// 最终伤害
int finalDamage = (int)(baseDamage * randomFactor);

// 暴击判定(10%概率)
if (Math.random() < 0.1) {
    finalDamage *= 1.5; // 1.5倍暴击
}

// 确保最小伤害为1
finalDamage = Math.max(1, finalDamage);
```

#### 法术伤害
```java
// 法术伤害 = 基础伤害 + (等级 × 倍率)
int spellDamage = spell.getBaseDamage() + (caster.getLevel() * spell.getLevelScale());

// 示例: 摄魂术(小龙女Lv43)
// 伤害 = 680 + (43 × 20) = 1540
```

### 3.4 战斗指令详解

#### 攻击 (Attack)
- 消耗: 无
- 效果: 对单个敌人造成物理伤害
- 目标: 单体

#### 法术 (Magic)
- 消耗: MP
- 效果: 根据法术类型
- 目标: 单体/全体/自身

#### 物品 (Item)
- 消耗: 物品数量
- 效果: 使用消耗品
- 目标: 根据物品类型

#### 逃跑 (Flee)
- 消耗: 无
- 成功率: 基于敏捷对比
- 限制: BOSS战无法逃跑

## 六、等级系统

### 6.1 经验值计算
```java
// 击败敌人获得经验
int expGained = enemy.getExpReward();

// 累计经验
currentExp += expGained;

// 升级判定
if (currentExp >= expToNextLevel) {
    levelUp();
}
```

### 6.2 升级效果
```java
void levelUp() {
    level++;
    
    // 属性提升(随机)
    hp += random(10, 20);
    attack += random(2, 5);
    defense += random(2, 4);
    agility += random(1, 3);
    
    // 学习新法术
    if (hasSpellToLearn(level)) {
        learnSpell(getSpellAtLevel(level));
    }
    
    // 恢复HP/MP
    hp = maxHp;
    mp = maxMp;
}
```

### 6.3 等级上限
- **哪吒**: 最高74级
- **其他角色**: 未明确上限

## 九、游戏平衡要点

### 9.1 难度曲线
```
章节 1-3:   简单 (熟悉系统)
章节 4-6:   中等 (角色成长)
章节 7-12:  较难 (需要策略)
章节 13-20: 困难 (需要练级)
```

### 9.2 经验值平衡
- 避免过度练级
- BOSS提供额外经验
- 支线任务奖励

### 9.3 物品掉落
- 小怪: 低掉率
- BOSS: 高掉率/固定掉落
- 宝箱: 固定物品
